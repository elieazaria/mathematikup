Fonction du Second Degré — Entreprise YaourtMada

📊 Cas d'entreprise

L'entreprise YaourtMada produit et vend des yaourts. Une étude de marché a montré que le bénéfice journalier dépend de la quantité produite. Le comptable a modélisé ce bénéfice par une fonction du second degré.

Bénéfice B(x) = -10x² + 600x - 5000
(x représente la quantité en litres, B(x) en Ariary)

💡 Détail des calculs

On remplace x par la valeur dans la formule B(x) = -10x² + 600x - 5000.

B(10) = -10(10)² + 600(10) - 5000 = -1000 + 6000 - 5000 = 0 Ariary
B(20) = -10(20)² + 600(20) - 5000 = -4000 + 12000 - 5000 = 3000 Ariary
B(30) = -10(30)² + 600(30) - 5000 = -9000 + 18000 - 5000 = 4000 Ariary
B(50) = -10(50)² + 600(50) - 5000 = -25000 + 30000 - 5000 = 0 Ariary

Analyse de la fonction

💡 Interprétation économique

Comme le coefficient a est négatif (-10), la parabole est orientée vers le bas. Cela signifie que la fonction admet un maximum.

Du point de vue économique, ce maximum correspond au bénéfice maximal. L'entreprise cherche donc les coordonnées de ce sommet pour savoir combien produire et combien elle gagnera.

3

Calcul du bénéfice maximal

Utilisez la formule du sommet : xS = -b / (2a) pour trouver la quantité optimale.

💡 Solution pas à pas

1. Identifier les coefficients : a = -10, b = 600, c = -5000.

2. Calculer l'abscisse du sommet (xS) :

xS = -b / (2a) = -600 / (2 × -10) = -600 / -20 = 30

L'entreprise doit produire 30 litres de yaourt.

3. Calculer le bénéfice maximal (ordonnée) :

B(30) = -10(30)² + 600(30) - 5000
B(30) = -9000 + 18000 - 5000
B(30) = 4000 Ariary

✅ Conclusion : Pour maximiser son profit, YaourtMada doit produire 30 litres, ce qui lui rapportera 4000 Ariary de bénéfice par jour.
