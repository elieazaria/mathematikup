Principe multiplicatif

Si on a n choix pour une premiere decision et m choix pour une seconde, le total est n x m.

4 x 5 x 3 = 60
n!

Factorielle

Le nombre de facons d'ordonner n objets est n! (n factorielle).

3! = 3 x 2 x 1 = 6
